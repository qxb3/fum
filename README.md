## Build from source

```bash
git clone https://github.com/qxb3/fum.git
cd fum
cargo build --release
# Either copy/move `target/release/yum` to /usr/bin
# Or add the release path to your system's path
# Moving fum binary to /usr/bin
mv target/release/fum /usr/bin
```
